# 「Invisipokky （インビジポッキー）』完全公式仕様書【最新・決定版】
> 姿の見えない殺人鬼1名と、ダッシュを禁じられた人間たちが、\
> 音と固有のガジェットを駆使して暗い館から脱出を目指す、命がけの非対称型・隠密ホラーケイドロゲーム。

**本作はチェストによる運要素を完全に廃止し、開幕の「役職固定ビルド」による戦略性と、統合版（マルチプラットフォーム）での完全再現性を重視した設計となっています。**

## 2. ゲームの流れと勝利条件
### ゲームの流れ
ゲーム開始の瞬間、全員に役職がランダム（または選択）で割り振られ、**対応する固定アイテム・ガジェットがインベントリに一瞬で自動支給されます。**\
人間は最初から動いて館に散らばることができますが、殺人鬼は最初の15秒間身動きが取れません。
### 勝利条件
* **人間**:
> 館に隠された「3つの鍵」をすべて解除し、開放された「脱出ゲート」から*1人でも逃げ切れば勝ち*。

> [!NOTE]
> 勝利のために全員が生存する必要はありません

* **殺人鬼**:
> 人間が脱出する前に、*全員を仕留めて全滅させれば勝ち*。

## 3. 鍵の解除による「殺人鬼の弱体化＆ラスト覚醒」
人間が鍵を開けるたびに殺人鬼の五感は奪われていきますが、最後の鍵が開いた瞬間に「**一撃必殺の怪物**」へと変貌します。

| 鍵の解除数 | 殺人鬼の視界変化（弱体化） | 殺人鬼の攻撃力 | 人間側の立ち回り |
----|----|----|----
| **初期（0個）** | **完全暗視**（人間も間取りも丸見え） |  2回殴ったらダウン | 鬼に見られている前提で隠密一 |
| **1個目** | **暗視没収**（館本来の暗闇になる） | 変化なし | 影に隠れてやり過ごせる |
| **2個目** | **時々、数秒間「盲目（真っ暗）」** になる | 変化なし | 鬼の盲目の隙を突いて動く！ |
| **3個目** | **常時「盲目」**（音と赤ピンだけで素敵） | **一撃必殺（ワンパン即死）** | 音を絶対に立てずにゲートへ！ |

## 4. 瀕死（ダウン）と救助システム
人間は1回殴られても即退場にはならず、「瀕死状態」として現場に踏みとどまり、味方に助けを求めることができます
> [!WARNING]
> 3個目の鍵が開いた後はワンパン即死のため発動しません。

* **殺人鬼が人間に殴ると**
  * 殺人鬼へのデバフ ... 3秒間「盲目」が強制付与され、ターゲットを一時的に見失う。
  * 人間へのデバフ ... 「移動速度低下（極大）」が付与され、地面を這いつくばる速度になる。

* **瀕死状態からの人間の状態変化**

| 秒数 | 状態 |
----|----
| 殴られて15秒間 | 血（赤ピン）は非表示になる潜伏タイム |
| 15秒経過後 | 血（赤ピン）が表示されるようになり、壁越しで位置が分かるようになる |
| 160秒経過後 | 大量出血による死亡、観戦者に |

### 救助の仕組み
他の仲間が瀕死の人間の横で5秒間スニーク（しゃがむ）すれば、救助が成功し通常状態に戻れる。\
救助された人間は、次に殴られたら即死亡（2回目で終わり）。

## 5. 音のペナルティ
人間が以下の行動をとると、その座標に1秒間だけ「赤色のパーティクル（赤ピン）」が発生し、壁越しでも殺人鬼に位置がバレます。

* ドアを開ける
* きしむ床の上を歩く
* アイテムの効果などによるダッシュ
> [!NOTE]
> ダッシュし続ける限りパーティクルが発生します
